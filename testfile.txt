This is sample file of git

git init
    --> Initiates git repo
The wokflow is:
working area --> staging area --> committed files

git add testfile.txt
    --> copies the files from working area to staging area

git commit -m "moving files to committed area" testfile.txt
    --> commits the files into committed area from staging one

git branch vp
    --> creats new branch

git checkout vp
    --> switch to existing branch

git checkout -b vp
    --> This creats new branch and switch to it automatically.

git branch -d vp
    -->Delete the branch

git branch
    --> Lists all the branch

git remote add origin location

git remote -v
    --> lists all the remote locations

git push origin master
    --> Pushes the branch to the remove
